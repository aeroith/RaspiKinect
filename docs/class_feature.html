Compute Fast Point Feature Histograms. The default FPFH implementation uses 11 binning subdivisions (e.g., each of the four feature values will use this many bins from its value interval), and a decorrelated scheme which results in a 33-byte array of float values. These are stored in a pcl::FPFHSignature33 point type.
