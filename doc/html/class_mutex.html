void Mutex::lock ( )
inline

If the mutex isn't currently locked by any thread, the calling thread locks it (from this point, and until its member unlock is called, the thread owns the mutex).

§ unlock()

void Mutex::unlock ( )
inline

Unlocks the mutex, releasing ownership over it. If other threads are currently blocked attempting to lock this same mutex, one of them acquires ownership over it and continues its execution.
